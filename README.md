# Grid07 Backend Assignment - Core API & Guardrails

## Project Overview
This is a Spring Boot microservice that acts as an API gateway with Redis-based guardrails for AI bot interactions. It handles concurrent requests, manages distributed state using Redis, and implements event-driven scheduling.

## Tech Stack
- Java 17
- Spring Boot 3.2.5
- PostgreSQL 15 (Docker)
- Redis 7 (Docker)
- Maven
- H2 Database (for testing without Docker)

---
